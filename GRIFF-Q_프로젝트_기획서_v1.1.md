## 4) 창고/재고/스튜디오 장비 관리
### 4-1. 재고 관리
- 재고 수량 추적
- 입고/출고 기록
- 보관 위치 관리

### 4-2. 장비 관리
- 장비 예약(시간/담당자/프로젝트)
- 장비 상태 체크(정상/고장/정비중)
- 정비 이력(일시, 내용, 담당자)

### 4-3. 프로젝트 연동
- 프로젝트별 장비 세트 템플릿
- 세팅/회수 체크리스트
- 확인 기록(담당자/시간/서명 또는 체크 로그)

### 4-4. QR 기반 운영
- 재고/장비별 고유 QR 발급
- QR 스캔 즉시 조회(자산명, 위치, 상태, 최근 이력)
- 멀티 스캔 일괄 처리(입고/출고/이동)
- 세트 단위 체크인/체크아웃 + 누락 자동 검출
- 스캔/변경 로그 저장(누가/언제/어디서/무엇을)

### 4-5. 장비 충전/정비 루틴 자동화
- 장비별 충전/정비 주기 설정 (예: 30일)
- 마지막 처리일 기준 다음 예정일 자동 계산
- 예: 2026-02-28 충전 완료 → 2026-03-28 충전 예정 알림
- D-3, D-1, 당일 알림 + 미처리 에스컬레이션
- 완료 시 다음 주기 자동 갱신, 지연 사유 기록

### 4-6. 보관위치 관리 상세
#### 4-6-1. 위치 체계 표준
- 위치 체계: 센터 > 구역 > 랙 > 선반 > 칸
- 예시: `STUDIO-A > A구역 > R02 > S03 > B04`
- 위치코드 표준: `A-R02-S03-B04` (고유)

#### 4-6-2. 위치 마스터 정보
- 필수 항목: 위치코드, 위치명, 상위위치코드, 위치타입, 수용량, 활성여부
- 위치타입: 창고/랙/선반/촬영장/임시보관/정비구역/외부반출
- 트리 구조로 상하위 위치 탐색 지원

#### 4-6-3. 자산 이동 처리 규칙
- 자산은 항상 1개의 현재 위치만 유지
- 이동 표준 절차: 출발 위치 스캔 → 자산 QR 스캔 → 도착 위치 스캔
- 이동 이력 자동 저장: 담당자, 시각, 출발/도착 위치, 사유

#### 4-6-4. 상태별 배치 제약
- 정비중 장비는 정비구역에만 배치 가능
- 대여중 장비는 외부위치(프로젝트/담당자)로 표시
- 분실 장비는 마지막 위치 및 마지막 스캔 시각 고정 저장

#### 4-6-5. 위치 관리 화면(운영 필수)
1. 위치 맵 뷰(트리/평면)
2. 빈 위치 찾기(수용량 기준)
3. 위치별 재고 목록
4. 최근 이동 로그
5. 세트 누락 탐지

#### 4-6-6. 알림 정책
- 위치 미확정 자산 24시간 경과 시 알림
- 금지 위치 배치 시 즉시 경고
- 장기 미이동 자산(예: 60일) 점검 알림

## 9) 프로젝트 (D)
### 9-1. 화면 구성
1. 프로젝트 현황판
2. View 모드: List / Board / Kanban / Calendar / Gantt / Node
3. 프로젝트 진행률
4. 워크로드

### 9-2. 주요 기능
- 프로젝트 추가
- Task 추가

### 9-3. 프로젝트별 R&R(Role & Responsibility) 관리
- 프로젝트 생성 시 역할 템플릿 선택(예: 일반 프로젝트 / 이벤트·행사)
- 역할별 책임 범위 정의: PM, 기획, 디자인, 영상, 운영, 올라운더
- 담당자 지정 시 권한/체크리스트/보고 라인 자동 연결
- 대체 담당자(백업) 지정 및 부재 시 자동 위임

### 9-4. 업무/이벤트 자동 임무카드 발급 (프로젝트 종류별)
- 프로젝트 생성 시 종류 선택에 따라 임무카드 자동 생성
  - 예: 일반 업무형 / 이벤트·행사형 / 콘텐츠 제작형 / 운영·유지보수형
- 임무카드는 프로젝트 템플릿 기반으로 기본 세트 자동 배정 + 담당자 매핑

#### 9-4-1. 공통 카드 필드
- 임무명, 목적, 담당자, 우선순위, 마감기한, 완료 기준, 체크포인트, 첨부자료
- 카드 상태: 대기 / 진행중 / 검토중 / 완료 / 이슈
- 수신자 확인(숙지) 버튼 필수
- 미숙지/미확인 상태는 시작 전 리마인드 + PM 알림

#### 9-4-2. 프로젝트 종류별 카드 정책
- 일반 업무형
  - 기획/제작/검수/배포 기본 카드 자동 생성
  - 승인 단계 카드(PM, 기획자) 자동 포함
- 이벤트·행사형
  - 사전준비/현장운영/철수정리 카드 자동 생성
  - 시간대/장소/장비체크/비상대응 체크포인트 포함
  - 행사 당일 상태 실시간 갱신(대기/진행중/완료/이슈)
- 콘텐츠 제작형
  - 아이디어/스크립트/촬영/편집/검수/게시 카드 자동 생성
  - 파일 버전 및 리뷰 라운드 체크포인트 포함
- 운영·유지보수형
  - 점검/이슈처리/복구/리포트 카드 자동 생성
  - SLA(응답/해결 목표시간) 및 에스컬레이션 규칙 포함

#### 9-4-3. 자동화 규칙
- 데일리 스크럼에서 추출된 액션아이템은 관련 임무카드에 자동 연결
- 일정 지연 발생 시 카드 마감 재산정 + 담당자 재배정 제안
- 장비 필요 업무는 장비 예약/보관위치/QR 체크와 자동 연동

### 9-5. 진행률 지표 정책 (제안 반영)
- **프로젝트 진행률**: Task 가중치 기반(기본 가중치: 일반 1, 핵심 2, 마일스톤 3)
- 계산식: `완료 가중치 합 / 전체 가중치 합 × 100`
- **업무 달성도(개인/팀)**와 분리 운영
  - 업무 달성도: 일/주 단위 목표 대비 완료율
  - 프로젝트 진행률: 프로젝트 단위 완료율

## 15) MVP 범위 제안 (1차 출시)
1. 기본 워크플로우 + 담당자 알림/승인
2. 프로젝트 관리(List/Board/Kanban/Calendar/Gantt/Node)
3. Task 관리(All/Today/Upcoming/Completed)
4. 데일리 스크럼(대화형) + Task 자동 등록
5. 재고/장비 + QR + 멀티체크 + 누락 검출
6. 장비 충전/정비 루틴 알림
7. 대시보드(오늘의 흐름 핵심 위젯)
8. 퍼포먼스(전체 공개 + 워크로드 상세)

## 16) 확정 사항 기록
1. 노드/칸반/캘린더/간트 모두 프로젝트 뷰에 위치
2. 워크로드: 대시보드 요약 + 퍼포먼스 상세 이중 구조
3. 진행률: 프로젝트 진행률(가중치)과 업무 달성도(목표 대비) 분리
4. 금융 탭: 추후 직접 정의
5. 퍼포먼스: 전체 공개
6. 온보딩: 정직원 대상
