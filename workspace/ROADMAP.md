## v0.3 — 기존 모듈 고도화

> v0.2에서 이관된 작업 + 견적서 고도화. 재정리 필요.

| 기능 | 설명 | 비고 |
|---|---|---|
| 프로젝트 메뉴 리워크 | 견적서/회의록/회고/입금 연결 해제, 다른 용도로 재설계 | → v0.5 프로젝트 모듈로 통합 검토 |
| 결제 대량전송 | 복수 항목 선택 → 은행 대량전송용 엑셀 다운로드 | 사이드바 비활성 처리 완료 |
| 계산서 발행 조회 | 외부 API 연동 세금계산서 발행 여부 확인 | 사이드바 비활성 처리 완료 |
| 견적서 워크플로우 | 심플 견적(초안) → 동시편집(최종) 흐름 연결 | |
| 견적서 템플릿 고도화 | 카테고리, 즐겨찾기 등 템플릿 관리 강화 | |
| 견적서 DB 불러오기/비교 | 기존 견적서에서 항목 불러오기 + 버전 간 금액 비교 뷰 | |
| 견적서 오류체크 | 필수 항목 누락, 단가 0원, 중복 항목명 등 자동 검증 | |
| 뉴스 큐레이션 | AI/Design/Tools 3카테고리 뉴스 수집·정리 | |
| 클라이언트 포털 | 외부 클라이언트 전용 견적서 확인/승인 공개 링크 | |
