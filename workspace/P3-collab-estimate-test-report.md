## 3. 발견된 이슈 및 수정사항

### 수정 완료
| 이슈 | 원인 | 수정 |
|---|---|---|
| RealtimeEstimateEditor에 specification/unit 컬럼 참조 | DB에 해당 컬럼 미존재 | 인터페이스 및 테이블에서 제거 |
| addRow에서 specification/unit INSERT 시도 | DB 스키마 불일치 | insert payload에서 제거 |
| Supabase FK join이 배열로 반환 | TypeScript 타입 불일치 | users 타입을 union으로 수정 |

## 4. Supabase Realtime 기능 현황

| 기능 | 상태 | 설명 |
|---|---|---|
| postgres_changes (UPDATE) | 구현됨 | 다른 사용자의 셀 변경 실시간 반영 |
| postgres_changes (INSERT) | 구현됨 | 행 추가 실시간 반영 |
| postgres_changes (DELETE) | 구현됨 | 행 삭제 실시간 반영 |
| Presence (접속자 표시) | 구현됨 | 아바타 + 이름 표시 |
| Presence (편집 셀 추적) | 구현됨 | 다른 사용자가 편집 중인 셀에 컬러 보더 표시 |
| 변경 하이라이트 애니메이션 | 구현됨 | 원격 변경 시 1.5초간 노란색 배경 |
| Debounce 자동 저장 | 구현됨 | 500ms 후 DB 저장 |
