# GRIFF-Q v0.3D — 통합 + QA

> v0.3 핵심 업무 엔진의 **마무리 Phase**
> **상위 계획서:** `GRIFF-Q-0.3-plan.md`
> **선행 의존:** v0.3A + v0.3B + v0.3C (전체 모듈 구현 완료)

---

## 1) 목표

v0.3A~C에서 구축한 5개 모듈(프로젝트/TASK/킥오프/스케줄/스크럼)을
**대시보드와 네비게이션에 통합**하고, **전체 QA + 회귀 테스트**로 마무리한다.

### 서브 작업

| 작업 | 설명 |
|---|---|
| **D1. 대시보드 위젯** | 스크럼/Task/프로젝트/일정 위젯 4종 추가 |
| **D2. 사이드바 업데이트** | 새 메뉴 5개 노출 + 네비게이션 정리 |
| **D3. 전체 QA** | 4조합 테스트 + 회귀 + 문제 0건까지 반복 |

## 2) 대시보드 위젯 (D1)

### 추가 위젯 4종

| 위젯 | 내용 | 데이터 소스 |
|---|---|---|
| **스크럼 상태** | 오늘 스크럼 완료/미완료 + 바로가기 | daily_scrums |
| **오늘의 Task** | 오늘 마감 Task 목록 (최대 5건) | tasks (due_date=today) |
| **프로젝트 진행률** | 활성 프로젝트 진행률 바 | projects (status=active) |
| **오늘 일정** | GRIFF 스케줄 오늘 일정 요약 | schedules |

### 대시보드 레이아웃 변경

현재 Bento 그리드에 4종 위젯을 추가. 기존 위젯(알림, 최근 활동, 통계 등)과 조화.

## 4) 전체 QA (D3)

### 테스트 루틴 (문제 0건까지 반복)

```
[Step 1] 자동 테스트
     │  · next build — 빌드 에러 0건
     │  · next lint — lint 에러 0건
     │
     ▼
[Step 2] 브라우저 직접 테스트 (Chrome DevTools MCP)
     │  · 라이트 × 데스크톱
     │  · 다크 × 데스크톱
     │  · 라이트 × 모바일 (375px)
     │  · 다크 × 모바일 (375px)
     │  · 스크린샷 캡처 + 클릭 테스트
     │  · 콘솔 에러 확인
     │
     ▼
[Step 3] 이슈 발견?
     │
     ├─ YES → [수정] → Step 1로 돌아감
     └─ NO → PASS
```

### 전체 시나리오 검증

#### 신규 기능

| 시나리오 | 검증 항목 |
|---|---|
| 대시보드 스크럼 위젯 | 오늘 스크럼 상태 + 바로가기 링크 |
| 대시보드 Task 위젯 | 오늘 마감 Task 목록 (최대 5건) |
| 대시보드 프로젝트 위젯 | 활성 프로젝트 진행률 바 |
| 대시보드 일정 위젯 | 오늘 일정 요약 |
| 사이드바 메뉴 | 5개 신규 메뉴 노출 + 정상 네비게이션 |
| 사이드바 모바일 | 햄버거 메뉴에서 모든 메뉴 접근 |

#### 회귀 테스트 (기존 기능)

| 모듈 | 검증 항목 |
|---|---|
| 회고 | 회고 CRUD + 점수 시스템 |
| 회의록 | 회의록 CRUD + 액션아이템 |
| 입금/결제 | 결제 CRUD + Excel 다운로드 |
| 견적서 | 심플 견적 + 동시견적 확인 |
| 클라이언트 예측 | Gemini 예측 정상 |
| 설정 | 알림 설정 정상 |
| 인증 | 로그인/로그아웃/세션 |

#### 다크모드 전체

- 모든 신규 페이지에서 다크모드 색상/대비/가독성 확인
- shadcn/ui 컴포넌트 다크 변환 정상

#### 모바일 전체

- 모든 신규 페이지에서 375px 레이아웃 정상
- 터치 인터랙션 (드래그, 스와이프) 정상
