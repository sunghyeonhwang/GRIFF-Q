# GRIFF-Q v0.4 — 대시보드 개편 + 세팅

> v0.3 핵심 업무 엔진 위에서 대시보드를 업무 중심으로 재설계
> **상태: 초안 (2026-03-05)**

---

## 0) 배경

v0.3에서 프로젝트/TASK/스크럼/스케줄이 추가되면서, 기존 대시보드(v0.25)는 더 이상 충분하지 않다.
업무 흐름 전체를 한눈에 파악할 수 있는 **탭 기반 대시보드**로 개편한다.

---

## 1) 범위

### 대시보드 개편

- **탭 전환 구조**: 업무 탭 / 금융 탭 (추후 확장 가능)
- **위젯 8종** (업무 탭):

| # | 위젯 | 설명 | 데이터 소스 |
|---|------|------|-------------|
| 1 | **오늘의 흐름** | 미션클리어, 놓친업무, Total Task, 진행중 | tasks, daily_scrums |
| 2 | **업무 달성도** | 업무 완료율/달성 현황 (개인/팀) | tasks |
| 3 | **리마인더** | 알림/리마인드 목록 | notifications (신규) |
| 4 | **내 업무** | 나에게 할당된 Task 목록 (상위 5건) | tasks |
| 5 | **데일리 스크럼** | 오늘 스크럼 상태 + 요약 | daily_scrums |
| 6 | **프로젝트 진행률** | 활성 프로젝트별 진행률 바 | projects, tasks |
| 7 | **워크로드** | 팀원별 업무 치중도 분석 | tasks |
| 8 | **오늘 미팅** | 오늘 예정 회의/일정 | schedules, meetings |

- **금융 탭**: 기존 대시보드 위젯 유지 (입금 추이, 견적 상태, 마감 임박)

### 세팅 (개인정보)

| 기능 | 설명 |
|---|---|
| 이름 수정 | 프로필 이름 변경 |
| 비밀번호 변경 | 현재 비밀번호 확인 → 새 비밀번호 설정 |
| 프로필 이미지 | 아바타 업로드 (선택) |
| 알림 설정 | 이메일/푸시 알림 on/off |

## 2) 기술 스택 추가

|---|---|---|
| 알림 | Supabase Realtime 또는 polling | 리마인더/알림 시스템 |

---

## 3) DB 스키마 (예상)

**notifications 테이블 (신규)**

| 컬럼 | 타입 | 설명 |
|---|---|---|
| `id` | uuid | PK |
| `user_id` | uuid | FK → users |
| `type` | enum | `reminder` / `task_assigned` / `deadline` / `scrum` / `system` |
| `title` | text | 알림 제목 |
| `message` | text | 알림 내용 |
| `link` | text | 클릭 시 이동 URL |
| `is_read` | boolean | 읽음 여부 |
| `created_at` | timestamptz | 생성 시점 |

## 4) 페이지 구성 (예상)

```
/dashboard                          # 대시보드 (탭 전환)
/dashboard?tab=work                 # 업무 탭 (기본)
/dashboard?tab=finance              # 금융 탭
/settings/profile                   # 개인정보 수정
/settings/notifications             # 알림 설정
```
